Detailed Description

template<class MutexType, class ConditionType, template< class > class RAIILockType, template< class > class DequeType>
class BufferedFileWriter< MutexType, ConditionType, RAIILockType, DequeType >

Class that performs writing to file
Writing is done in buffered(chunked) mode:
Takes a buffer from the queue and write it to the output file

Template Parameters
MutexTypeData type used as a mutex (std::mutex/boost::interprocess::mutex)
ConditionTypeData type used as a conditional variables(std::conditional_variable/boost::interprocess::condition)
RAIILockTypeData type used as a RAII lock (std::unique_lock/boost::interprocess::scoped_lock)
DequeTypeData type used as an inner deque types (std::deque/boost::interprocess::deque)
