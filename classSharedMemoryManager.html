Detailed Description

Class handles:
all allocations and deallocations done in shared memory:

  • Queue that chunks read from files
  • Process Info
    allocation and deallocation of shared memory region
    determines if the process is reader (first process creates shared memory region) or writer (shared memory region already exists)
