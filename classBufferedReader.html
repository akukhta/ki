Detailed Description

template<class Tag>
class BufferedReader< Tag >

Class that performs reading from file
Reading is done in buffered(chunked) mode:
file is being read in small buffers/chunks and those buffers are being stored in the queue
Later on those buffers will be used to write data from
Reading is done with direct accessing of the io buffer (for more info check "Linux System Programming" by Robert Love, Chapter 3, pg 123)
