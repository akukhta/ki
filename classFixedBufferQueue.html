Detailed Description

template<class Tag>
class FixedBufferQueue< Tag >

Queue that holds buffers
Provides following functionality:

  1. Get free buffer to read data into
  2. Get filled buffer to write data from
    Note: Once buffer is no longer neeeded, it should be returned back to the queue. See returnBuffer function
    P.S. not inherited from IQueue because IPC queue would store virtual function pointer And this pointer wouldn`t work from another process, if we use queue allocated withing shared memory
